EPUB 3 Accessibility Guidelines

IDPFによる公式のガイドライン。EPUB Accessibility 1.0 が公開されるかなり前からあり、位置づけは曖昧になっている。更新はとまっているが、説明がそれなりに詳細なので上に情報がない場合は参考になる。

EPUB Accessibility 1.0 日本語訳

W3C（以前はIDPF）公式のアクセシビリティ仕様。

EPUB Accessibility Techniques 1.0 日本語訳

EPUBにおける、WCAGで言うところの達成方法集に相当するもの。

How to Meet WCAG 2 (クイックリファレンス)

WCAGのある達成基準を満たすためにどういう実装方法を選択すればいいか、またどういった失敗例があるかを参照できるクイック・リファレンス。ここから解説書や達成方法集の個別リンクに飛べるので、WCAG仕様を読みこなす上で便利です。

WCAG 2.0 達成方法集 日本語訳

W3Cが公開しているTechniques for WCAG 2.0の日本語訳。

WCAG 2.0とUnderstanding WCAG 2.0は、「技術に依存しない」文書であるのに対し、Techniques for WCAG 2.0は、具体的なHTMLやCSSのコード、チェック方法を例示しつつ、どのように実装すべきか参考になる文書を提供する。

WCAG 2.0 解説書 日本語訳

WCAG 2.0の理解と実装を助けるためにW3Cが公開している文書であるUnderstanding WCAG 2.0の日本語訳。

抽象度の高いWCAG 2.0にたいして、

  • WCAG 2.0の達成基準がどのような意図で書かれているのか
  • WCAG 2.0の達成基準の中で用いられている重要な用語
  • WCAG 2.0の達成基準が、どんな種類の障害にたいして、どのように役に立つのか
  • どのように対応すべきかの参考資料への動線（Techniques for WCAG 2.0へのリンク）
  • まずい対応の事例（よくある失敗例）

などを説明する文書。いわばWCAG 2.0の副読本。

Web Content Accessibility Guidelines (WCAG) 2.0 日本語訳

W3Cが勧告したWCAG 2.0の日本語訳。ウェブアクセシビリティ基盤委員会（WAIC）がメンテナンスを行っている。

この文書単体ではあまり役に立つものではなく、WCAG 2.0 解説書やWCAG 2.0 達成方法集とあわせて読むようにできている。

日本のウェブアクセシビリティの産業規格であるJIS X 8341-3:2016は、本仕様を基に策定されている。

Web Content Accessibility Guidelines (WCAG) 2.1 日本語訳

W3Cが勧告したWCAG 2.1の日本語訳。ウェブアクセシビリティ基盤委員会（WAIC）がメンテナンスを行っている。

WCAG 2.0に加えて、「文字キーのショートカット」や「ポインタのジェスチャ」などの達成基準が追加されている。

日本のウェブアクセシビリティの産業規格であるJIS X 8341-3:2016は、本仕様でなくWCAG 2.0を基にしている。

まだ「WCAG 2.1 解説書」や「WCAG 2.1 達成方法集」は翻訳されていないので、「早く日本語で読みたい！」というひとはウェブアクセシビリティ基盤委員会の翻訳チームに参加すると、幸せになれるかもしれない。
